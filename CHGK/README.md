CHGK

Клиент для игры "Что?Где?Когда?"

Работает с базой вопросов "Что?Где?Когда?" (http://db.chgk.info/)

Саму базу можно взять с http://bilbo.dynip.com/home/cvsroot (раздел ""Секрет" третьий" в описании формата)
